# 音频算法系统补强计划

这是一个纯静态 GitHub Pages 学习网页，不需要后端服务。

## 发布文件

- `index.html`
- `styles.css`
- `app.js`
- `lessons.js`
- `lesson-extras.js`
- `.nojekyll`

## GitHub Pages 设置

在仓库 `Settings -> Pages` 中选择：

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/root`

保存后等待 1-3 分钟即可访问 GitHub Pages 链接。
